-El proyecto está dividido en dos partes, el frontend (carpeta prueba-mayangps) y el backend (carpeta server).

-Para correr el frontend será necesario tener instalado Angular y simplemente instalar los paquetes necesarios con npm install para correr el comando ng serve
y que nos muestre la aplicación en el navegador.

-Para correr el backend solo es necesario correr el comando npm install y npm run dev, que es el script que inica el servidor

-El script de la base de datos está en la carpeta DB, solo se corre ese archivo en mysql y creará la base de datos.
 
-Las credenciales para la base de datos están en un archivo llamado 'keys.ts' donde pueden ser cambiadas según sea necesario.
